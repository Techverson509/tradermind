# TraderMind — Goal & Habit Tracker 🧠

A professional Goal & Habit Tracker designed for traders.
Available in 3 languages: Haitian Creole, French & English.

## Features

- 📅 Daily habit tracking with streak counter
- 🎯 Trading goals with progress rings (Profit, Win Rate, Pips...)
- 📔 Personal journal — write notes every day
- 🌙 Dark / Light theme — follows your system automatically
- 🌍 Trilingual — HT / FR / EN
- 💾 Data saved locally — works offline
- 📱 PWA — installable on Android & iPhone

## Install on Phone

**Android (Chrome):**
1. Open the URL in Chrome
2. Tap ⋮ → "Add to Home Screen"

**iPhone (Safari):**
1. Open the URL in Safari
2. Tap Share → "Add to Home Screen"
